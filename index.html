Introdução ao Html

(Voltar)

HTML é a linguagem de marcação mais utilizada na criação de páginas da web. É uma linguagem simples que permite aos desenvolvedores criar conteúdo estruturado e semântico para a web. HTML é a sigla em inglês para "Hypertext Markup Language", que significa "Linguagem de Marcação de Hipertexto". Com HTML, você pode criar uma estrutura básica para a sua página, incluindo cabeçalhos, parágrafos, listas, links e imagens. A linguagem é composta por elementos e atributos, que são usados para definir o conteúdo e a aparência do seu site.

Tags

(Voltar)

As tags são um dos principais componentes da linguagem HTML. Elas são usadas para definir os elementos que compõem uma página da web, como cabeçalhos, parágrafos, listas e imagens. Cada tag é composta por um nome e um conjunto de atributos, que são usados para definir as propriedades do elemento.

exemplos de tags: h1, title, head, a, ul, ol

Atributos

(Voltar)

les podem ser usados para definir as propriedades dos elementos. Atributos são usados para definir as características de um elemento em uma página da web, como a cor, tamanho, URL de um link, etc.
